Unzip JBox.zip archive.

The easiest way:

```

```
2) Go to http://localhost:port/

Also, you can use apache2 to host this task.

1) Copy progect folder to /var/www/html folder.

2) Run next comand: 
```
sudo service apache2 start
```
3) Go to http://localhost/taskname/
